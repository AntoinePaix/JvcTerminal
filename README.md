# JvcTerminal
Script Python très simple qui affiche les différents topics d'une page provenant des forums de jeuxvideo.com

## Utilisation

    python3 jvc-terminal -u http://www.jeuxvideo.com/forums/0-47-0-1-0-1-0-programmation.htm
